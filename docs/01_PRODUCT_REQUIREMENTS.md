# Product Requirements Document (PRD)

### Authentication

- Register
- Login
- Logout
- Google OAuth
- Forgot Password
- Email Verification
- JWT Authentication
- Refresh Tokens

---

### User Profile

- Avatar
- Name

- Notification Preferences

- Reviews

### Booking

- Split Payments
- Refunds
- Invoices
- Receipts

### Chat

- Private Chat

- Read Receipts

### Notifications

- Email
